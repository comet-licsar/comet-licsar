subset_initialise_corners(frame, lon1, lon2, lat1, lat2, sid, is_volc=False, resol_m=30)[source]

This will initialise a subset given by corner lon/lat-s. The results will be stored in $LiCSAR_procdir/subsets

Parameters:

  • frame (str) – frame ID,

  • lon1 (float, float) – corner longitudes,

  • lon2 (float, float) – corner longitudes,

  • lat1 (float, float) – corner latitudes,

  • lat2 (float, float) – corner latitudes,

  • sid (str) – string ID (for volcano, use its volcano ID number)

  • is_volc (bool) – if true, it will set the output folder $LiCSAR_procdir/subsets/volc

  • resol_m (float) – output resolution in metres to have geocoding table ready in (note, RSLCs are anyway in full res)

subset_initialise_centre_coords(frame, clon, clat, sid, is_volc=False, radius=12.5, resol_m=30)[source]

This will initialise a subset given by centre lon/lat and radius in km. The results will be stored in $LiCSAR_procdir/subsets

Parameters:

  • frame (str) – frame ID,

  • clon (float) – centre longitude,

  • clat (float) – centre latitude,

  • sid (str) – string ID (for volcano, use its volcano ID number)

  • is_volc (bool) – if true, it will set the output folder $LiCSAR_procdir/subsets/volc

  • radius (float) – radius (half of the diameter) of the subset scene, in km

  • resol_m (float) – output resolution in metres to have geocoding table ready in (note, RSLCs are anyway in full res)
